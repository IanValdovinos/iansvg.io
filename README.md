# iansvg.io

## Package Installations

### Basic

`pip install fastapi`

`pip install "uvicorn[standard]"`

`pip install python-multipart`

### Database

`pip install sqlalchemy`

`brew install sqlite`

`pip install psycopg2-binary`

`pip install alembic`

### Encryption

`pip install passlib`

`pip install bcrypt==4.0.1`

### JWT

`pip install "python-jose[cryptography]"`
